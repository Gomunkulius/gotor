<center><h1>GOTOR</h1> </center>

The least functional and simplest torrent client you've ever seen

## Install 

#### MacOS/Linux:
```shell
sudo .\install.sh
```
#### Windows:
Run with administrator rights:
```shell
.\install-win.bat
```
## Build

#### MacOS/Linux/Windows:
```shell
make build
```

#### Or
```shell
go build cmd/main/main.go
```

## Run tests
```shell
make test
```
